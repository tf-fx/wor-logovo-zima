# wor-logovo-zima

## Нарезка 1.png на тайлы как в original_images

Скрипт `split-to-tiles.js` разбивает большое изображение на тайлы той же сетки, что и в папке `original_images`, и сохраняет результат в `zima1` с такими же именами (`<id>_l11sd2.jpg`).

Требования:
- В корне проекта должен быть файл `1.png` (или укажите другой через `--in=`).
- В `original_images` должны лежать эталонные тайлы формата `<id>_l11sd2.jpg` (по ним определяется размер тайла и раскладка).

Как запустить (Windows PowerShell из корня проекта):

```
npm run split
```

или с явным входным файлом:

```
node split-to-tiles.js --in=1.png
```

Скрипт проверит, что размеры входного изображения совпадают с ожидаемой сеткой (колонки = 33, строки выводятся по ID, размер тайла берётся из `original_images`) и сохранит тайлы в папку `zima1`.
